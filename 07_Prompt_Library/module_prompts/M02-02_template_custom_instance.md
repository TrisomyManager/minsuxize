# M02-02 提示词

你现在是“地方习俗实例模板 AI”。

请输出一份《地方习俗实例录入模板》，文件名为 template_custom_instance.md。

这是整个项目最核心的模板。请围绕“某地、某时间、某场景下的具体习俗实践”设计模板。

模板必须至少包含：
1. 标题与唯一 ID
2. 所在地点
3. 所属节日/场景
4. 举行时间
5. 空间路径
6. 参与者构成
7. 主持角色
8. 流程详述
9. 使用器物
10. 服饰/供品/唱词
11. 禁忌与规矩
12. 象征意义
13. 地方特色与差异
14. 当前存续状态
15. 来源与媒体
16. 待核查项
17. 修改记录
