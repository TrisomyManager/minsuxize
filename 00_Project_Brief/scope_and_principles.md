# 范围与原则

## 1. 内容范围
- 岁时节令
- 人生礼仪
- 家族宗族
- 信仰祭祀
- 生产习俗
- 居住建筑相关习俗
- 饮食与供品习俗
- 口头传统与仪式语言
- 禁忌与规矩

## 2. 地域范围
首期只做试点区域，不做全国铺开。

## 3. 精度原则
地点应尽量精确到：
国家 → 省 → 市 → 县/区 → 乡镇/街道 → 行政村 → 自然村 → 空间点位

## 4. 事实原则
- 没有来源，不落事实结论。
- 推测必须显式标注为“待核查”。
- 同一习俗允许多种说法并存。

## 5. 执行原则
- 单次会话只完成一个模块。
- 模块成果必须可归档、可交接、可继续迭代。
- 每个模块必须写明：输入、输出、边界、未决问题。
